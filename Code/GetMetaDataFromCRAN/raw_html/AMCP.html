AMCP: A Model Comparison Perspective

Accompanies "Designing experiments and analyzing data: A model comparison perspective" (3rd ed.) by Maxwell, Delaney, & Kelley (forthcoming from Routledge). Contains all of the data sets in the book's chapters and end-of-chapter exercises. Information about the book is available at <http://www.DesigningExperiments.com>.
