rscala: Bi-Directional Interface Between R and Scala with Callbacks

The Scala <http://www.scala-lang.org/> interpreter is embedded in R and callbacks to R from the embedded interpreter are supported. Conversely, the R interpreter is embedded in Scala. Scala versions in the 2.10.x, 2.11.x, and 2.12.x series are supported.
