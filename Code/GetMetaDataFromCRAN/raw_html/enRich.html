enRich: An R Package for the Analysis of Multiple ChIP-Seq Data

An R package for joint statistical modelling of ChIP-seq data, accounting for technical/biological replicates, multiple conditions and different ChIP efficiencies of the individual experiments.

Depends: R (≥ 3.2.2), parallel
