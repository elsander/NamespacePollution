EffectStars: Visualization of Categorical Response Models

Notice: The package EffectStars2 provides a more up-to-date implementation of effect stars! EffectStars provides functions to visualize regression models with categorical response. The effects of the variables are plotted with star plots in order to allow for an optical impression of the fitted model.
