vegdata: Access Vegetation Databases and Treat Taxonomy

Handling of vegetation data from Turboveg (<http://www.synbiosys.alterra.nl/turboveg/>) and other sources (<http://www.vegetweb.de>). Taxonomic harmonization given appropriate taxonomic lists (e.g. the German taxonomical standard list "GermanSL", <http://www.botanik.uni-greifswald.de/GermanSL.html>).
