NetworkChange: Bayesian Package for Network Changepoint Analysis

Network changepoint analysis for undirected network data. The package implements a hidden Markov multilinear tenstor regression model (Park and Sohn, 2017, <http://jhp.snu.ac.kr/NetworkChange.pdf>). Functions for break number detection using the approximate marginal likelihood and WAIC are also provided.
