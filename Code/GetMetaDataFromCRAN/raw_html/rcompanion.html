rcompanion: Functions to Support Extension Education Program Evaluation

Functions and datasets to support "Summary and Analysis of Extension Education Program Evaluation in R" and "An R Companion for the Handbook of Biological Statistics". Vignettes are available at <http://rcompanion.org>.
