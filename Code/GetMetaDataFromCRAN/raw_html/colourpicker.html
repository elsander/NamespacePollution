colourpicker: A Colour Picker Tool for Shiny and for Selecting Colours in Plots

A colour picker that can be used as an input in Shiny apps or 'Rmarkdown' documents. A Plot Colour Helper tool is available as an 'RStudio' addin, which helps you pick colours to use in your plots. A more generic Colour Picker 'RStudio' addin is also provided to let you select colours for use in your R code.
