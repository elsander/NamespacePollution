edrGraphicalTools: Provides tools for dimension reduction methods

This package comes to illustrate the articles "A graphical tool for selecting the number of slices and the dimension of the model in SIR and SAVE approaches" and "Comparison of sliced inverse regression approaches for underdetermined cases"
