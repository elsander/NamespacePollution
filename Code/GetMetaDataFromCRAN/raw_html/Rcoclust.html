Rcoclust: Co-Clustering with Document-Term Matrix

Functions for co-clustering with C code illustrating two diagonal approaches, (Ailem & Role & Nadif, 2016, <doi:10.1016/j.knosys.2016.07.002>), (Laclau & Nadif, 2016, <doi:10.1016/j.neucom.2016.02.003>). A relevant initialization and a suitable setting of the parameters are required. For a dimensionality reduction via random projection, a function is included.
