grattan: Perform Common Quantitative Tasks for Australian Analysts and to Support Grattan Institute Analysis

A series of functions focused on costing and evaluating Australian tax policy in support of the Grattan Institute's Australian Perspectives program. For access to the taxstats package, please run install.packages("taxstats", repos = "https://hughparsonage.github.io/drat/", type = "source"). N.B. The taxstats package is approximately 50 MB.
