BCEE: The Bayesian Causal Effect Estimation Algorithm

Implementation of the Bayesian Causal Effect Estimation algorithm, a data-driven method for the estimation of the causal effect of an exposure on a continuous outcome. For more details, see Talbot et al. (2015) DOI:10.1515/jci-2014-0035.
