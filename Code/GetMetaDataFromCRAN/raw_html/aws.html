aws: Adaptive Weights Smoothing

Collection of R-functions implementing the Propagation-Separation Approach to adaptive smoothing as described in "J. Polzehl and V. Spokoiny (2006) <doi:10.1007/s00440-005-0464-1>" and "J. Polzehl and V. Spokoiny (2004) <doi:10.20347/WIAS.PREPRINT.998>".
