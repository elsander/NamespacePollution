BayClone2: Bayesian Feature Allocation Model for Tumor Heterogeneity

A Bayesian feature allocation model is implemented for inference on tumor heterogeneity using next-generation sequencing data. The model identifies the subclonal copy number and single nucleotide mutations at a selected set of loci and provides inference on genetic tumor variation.
