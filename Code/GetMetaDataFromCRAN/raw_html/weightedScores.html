weightedScores: Weighted Scores Method for Regression Models with Dependent Data

Has functions to implement the weighted scores method and CL1 information criteria as an intermediate step for variable/correlation selection for longitudinal categorical and count data in Nikoloulopoulos, Joe and Chaganty (2011, Biostatistics, 12: 653-665) and Nikoloulopoulos (2015a,2015b).
