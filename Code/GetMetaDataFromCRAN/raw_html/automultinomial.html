automultinomial: Autologistic and Automultinomial Spatial Regression and Variable Selection

Contains functions for autologistic variable selection and parameter estimation for spatially correlated categorical data (including k>2). The main function is MPLE. Capable of fitting the centered autologistic model described in Caragea and Kaiser (2009)<doi:10.1198/jabes.2009.07032>, as well as the traditional autologistic model of Besag (1974)<http://www.jstor.org/stable/2984812>.
