ngspatial: Fitting the Centered Autologistic and Sparse Spatial Generalized Linear Mixed Models for Areal Data

Provides tools for analyzing spatial data, especially non-Gaussian areal data. The current version supports the sparse spatial generalized linear mixed model of Hughes and Haran (2013) <doi:10.1111/j.1467-9868.2012.01041.x> and the centered autologistic model of Caragea and Kaiser (2009) <doi:10.1198/jabes.2009.07032>.
