haploR: Query HaploReg and RegulomeDB

A set of utilities for querying HaploReg <http://archive.broadinstitute.org/mammals/haploreg/haploreg.php> and RegulomeDB <http://www.regulomedb.org> web-based tools. The package connects to Haploreg or RegulomeDB, searches and downloads results, without opening web pages, directly from R environment. Results are stored in a data frame that can be directly used in various kinds of downstream analyses.
