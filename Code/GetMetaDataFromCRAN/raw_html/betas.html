betas: Standardized Beta Coefficients

Computes standardized beta coefficients and corresponding standard errors for the following models: linear regression models with numerical covariates only, linear regression models with numerical and factorial covariates, weighted linear regression models, all these linear regression models with interaction terms, and robust linear regression models with numerical covariates only.
