alineR: Alignment of Phonetic Sequences Using the 'ALINE' Algorithm

Functions are provided to calculate the 'ALINE' Distance between words. The score is based on phonetic features represented using the Unicode-compliant International Phonetic Alphabet (IPA). Parameterized features weights are used to determine the optimal alignment and functions are provided to estimate optimum values.
