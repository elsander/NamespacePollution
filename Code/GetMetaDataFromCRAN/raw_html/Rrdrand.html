Rrdrand: 'DRNG' on Intel CPUs with the 'RdRand' Instruction for R

Make use of the hardware random number accessed by the 'RdRand' instruction in recent Intel CPUs (Ivy Bridge and later). 'DRNG' is "Digital Random Number Generator".
