antaresViz: Antares Visualizations

Visualize results generated by Antares, a powerful software developed by RTE to simulate and study electric power systems (more information about Antares here: <https://antares.rte-france.com>). This package provides functions that create interactive charts to help Antares users visually explore the results of their simulations.
