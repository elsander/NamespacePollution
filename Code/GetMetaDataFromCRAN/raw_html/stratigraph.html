stratigraph: Toolkit for the plotting and analysis of stratigraphic and palaeontological data

A collection of tools for plotting and analyzing paleontological and geological data distributed through through time in stratigraphic cores or sections. Includes some miscellaneous functions for handling other kinds of palaeontological and paleoecological data.
