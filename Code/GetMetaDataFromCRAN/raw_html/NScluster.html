NScluster: Simulation and Estimation of the Neyman-Scott Type Spatial Cluster Models

Simulation and estimation for Neyman-Scott spatial cluster point process models and their extensions. For estimating parameters by the simplex method, parallel computation using OpenMP application programming interface is available.
