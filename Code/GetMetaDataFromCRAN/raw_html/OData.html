OData: R Helper for OData Web Services

Helper methods for accessing data from web service based on OData Protocol. It provides several helper methods to access the service metadata, the data from datasets and to download some file resources (it only support CSV for now). For more information about OData go to <http://www.odata.org/documentation/>.
