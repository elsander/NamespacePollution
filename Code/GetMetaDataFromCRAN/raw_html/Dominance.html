Dominance: ADI (Average Dominance Index), Social Network Graphs with Dual Directions, and Music Notation Graph

Can calculate ADI (Average Dominance Index), FDI (Frequency based Dominance Indexand) and can build social network graphs with dual directions, can build a Music Notation Graph.
