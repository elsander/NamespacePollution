SemiCompRisks: Hierarchical Models for Parametric and Semi-Parametric Analyses of Semi-Competing Risks Data

Parametric and semi-parametric analyses of semi-competing risks/univariate survival data. For semi-competing risks data, the package contains implementations of hierarchical models for independent data (Lee et al., 2015; <dio:10.1111/rssc.12078>) and cluster-correlated data (Lee et al., 2016; <dio:10.1080/01621459.2016.1164052>).
