RPyGeo: ArcGIS Geoprocessing in R via Python

Provide access to (virtually any) ArcGIS Geoprocessing tool from within R by running Python geoprocessing scripts without writing Python code or touching ArcGIS. Requires ArcGIS >=9.2, a suitable version of Python (for ArcGIS 9.2: Python 2.4; for ArcGIS 10.0: 2.6), and Windows.
