SubLasso: Gene selection using Lasso for Microarray data with user-defined genes fixed in model

The package implements a convenient procedure for microarray study, which is to do gene selection and classification simultaneously for binary outcomes. Users needn't to tune the parameters and can fix any genes that they desire to keep in the model. The K-folds cross validation results are returned.
