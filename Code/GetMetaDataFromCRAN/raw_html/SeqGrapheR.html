SeqGrapheR: Simple GUI for Graph Based Visualization of Cluster of DNA Sequence Reads

The SeqGrapheR package provide interactive GUI for visualization of DNA sequence clusters. Details and principles of usage are described in user manual and (2010 BMC Bioinformatics 11:378). For full functionality installed NCBI blast is required.
