sExtinct: Calculates the historic date of extinction given a series of sighting events

This package combines several sighting based estimators of historical extinction, allowing them to be run simultaneously or individually. Code for this package was contributed by Ben Collen, Gene Hunt and Tracy Rout. Additional code was taken from McPherson & Myers (2009).
