flowDiv: Cytometric Diversity Indices from 'FlowJo' Workspaces

Concatenates some 'flowWorkspace', 'flowCore', 'vegan' and 'gdata' packages functionalities to import 'FlowJo' workspaces and calculates ecological diversity indices for gated populations, based on two dimensions cytograms.
