JGR: JGR - Java GUI for R

Java GUI for R - cross-platform, universal and unified Graphical User Interface for R. For full functionality on Windows and Mac OS X JGR requires a start application which depends on your OS. This can be downloaded from JGR website: http://rforge.net/JGR/
