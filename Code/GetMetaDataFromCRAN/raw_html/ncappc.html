ncappc: NCA Calculation and Population PK Model Diagnosis

A flexible tool is presented here that can perform (i) traditional non-compartmental analysis (NCA) and (ii) simulation-based posterior predictive checks for a population pharmacokinetic (PK) model using NCA metrics.
