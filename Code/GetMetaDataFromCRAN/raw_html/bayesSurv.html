bayesSurv: Bayesian Survival Regression with Flexible Error and Random Effects Distributions

Contains Bayesian implementations of Mixed-Effects Accelerated Failure Time (MEAFT) models for censored data. Those can be not only right-censored but also interval-censored, doubly-interval-censored or misclassified interval-censored.
