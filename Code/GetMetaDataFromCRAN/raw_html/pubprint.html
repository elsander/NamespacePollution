pubprint: Printing Results of Statistical Computing in a Publishable Way

Takes the output of statistical tests and transforms it in a publish-friendly pattern. Currently only APA (American Psychological Association) style is supported with output to HTML, LaTeX, Markdown and plain text. It is easily customizable, extendable and can be used well with 'knitr'. Additionally 'pubprint' offers a memory system that allows to save and retrieve results of computations.
