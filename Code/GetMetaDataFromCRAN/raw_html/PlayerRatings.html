PlayerRatings: Dynamic Updating Methods for Player Ratings Estimation

Implements schemes for estimating player or team skill based on dynamic updating. Implemented methods include Elo, Glicko and Stephenson. Contains pdf documentation of a reproducible analysis using approximately two million chess matches.
