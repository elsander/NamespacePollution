APSIM: General Utility Functions for the 'Agricultural Production Systems Simulator'

Contains functions designed to facilitate the loading and transformation of 'Agricultural Production Systems Simulator' output files <https://www.apsim.info>. Input meteorological data (also known as "weather" or "met") files can also be generated from user supplied data.
