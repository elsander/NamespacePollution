HRM: High-Dimensional Repeated Measures

Methods for testing main and interaction effects in possibly high-dimensional repeated measures in factorial designs. The observations of the subjects are assumed to be multivariate normal. It is possible to use up to 2 whole- and 3 subplot factors.
