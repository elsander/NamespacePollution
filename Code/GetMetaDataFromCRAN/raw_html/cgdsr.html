cgdsr: R-Based API for Accessing the MSKCC Cancer Genomics Data Server (CGDS)

Provides a basic set of R functions for querying the Cancer Genomics Data Server (CGDS), hosted by the Computational Biology Center at Memorial-Sloan-Kettering Cancer Center (MSKCC).
