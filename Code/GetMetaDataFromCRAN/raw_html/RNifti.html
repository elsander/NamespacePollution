RNifti: Fast R and C++ Access to NIfTI Images

Provides very fast access to images stored in the NIfTI-1 file format <http://nifti.nimh.nih.gov/nifti-1>, with seamless synchronisation between compiled C and interpreted R code. Not to be confused with 'RNiftyReg', which provides tools for image registration.
