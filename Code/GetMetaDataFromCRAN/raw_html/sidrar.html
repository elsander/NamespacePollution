sidrar: An Interface to IBGE's SIDRA API

Allows the user to connect with IBGE's (Instituto Brasileiro de Geografia e Estatistica, see <http://www.ibge.gov.br/> for more information) SIDRA API in a flexible way. SIDRA is the acronym to "Sistema IBGE de Recuperacao Automatica" and is the system where IBGE turns available aggregate data from their researches.
