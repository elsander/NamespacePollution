GerminaR: Germination Indexes for Seed Germination Variables for Ecophysiological Studies

Different types of seed indexes, rates and visualization techniques are used to provide a robust approach for germination data analysis. The package aims to make available germination seed indexes and graphical functions to analyze germination seed data.
