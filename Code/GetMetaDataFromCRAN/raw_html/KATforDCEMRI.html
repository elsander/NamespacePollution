KATforDCEMRI: Kinetic analysis and visualization of DCE-MRI data

Package for kinetic analysis of longitudinal voxel-wise Dynamic Contrast Enhanced MRI data. Includes tools for visualization and exploration of voxel-wise parametric maps.
