textir: Inverse Regression for Text Analysis

Multinomial [inverse] regression inference for text documents and associated attributes. Provides fast sparse multinomial logistic regression for phrase counts. A minimalist partial least squares routine is also included. Note that the topic modeling capability of textir is now a separate package, maptpx.
