FinePop: Fine-Scale Population Analysis

Statistical tool set for population genetics. The package provides empirical Bayes estimator of Fst and other measures of genetic differentiation. It also provides interfaces to read and manipulate 'GENEPOP' format data files and allele/haplotype frequency format files.
