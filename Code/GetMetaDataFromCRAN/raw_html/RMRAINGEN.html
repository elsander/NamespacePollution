RMRAINGEN: RMRAINGEN (R Multi-site RAINfall GENeretor): a package to generate daily time series of rainfall from monthly mean values

This package contains functions and S3 methods for spatial multi-site stochastic generation of daily precipitation. It generates precipitation occurrence in several sites using Wilks' Approach (1998). Bugs/comments/questions/collaboration of any kind are warmly welcomed.
