FLSSS: Multi-Threaded Multidimensional Fixed Size Subset Sum Solver and Extension to General-Purpose Knapsack Problem

A novel algorithm for solving the subset sum problem with bounded error in multidimensional real domain and its application to the general-purpose knapsack problem.
