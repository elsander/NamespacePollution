bibliometrix: Bibliometric and Co-Citation Analysis Tool

Tool for quantitative research in scientometrics and bibliometrics. It provides various routines for importing bibliographic data from SCOPUS (<http://scopus.com>) and Thomson Reuters' ISI Web of Knowledge (<http://www.webofknowledge.com/>) databases, performing bibliometric analysis and building data matrices for co-citation, coupling, scientific collaboration and co-word analysis.
