DAAGxtras: Data Sets and Functions, supplementary to DAAG

various data sets used in additional exercises for the book Maindonald, J.H. and Braun, W.J. (3rd edn 2010) "Data Analysis and Graphics Using R", and for a 'Data Mining' course. Note that a number of datasets that were in earlier versions of this package have been transferred to the DAAG package.
