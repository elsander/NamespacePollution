equateIRT: Direct, Chain and Average Equating Coefficients with Standard Errors Using IRT Methods

Computation of direct, chain and average (bisector) equating coefficients with standard errors using Item Response Theory (IRT) methods for dichotomous items. Test scoring can be performed by true score equating and observed score equating methods.
