libsoc: Read, Create and Write 'PharmML' Standard Output (so) XML Files

Handle 'PharmML' (Pharmacometrics Markup Language) standard output (SO) XML files. SO files can be created, read, manipulated and written through a data binding from the XML structure to a tree structure of R objects.
