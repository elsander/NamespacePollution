egonet: Tool for ego-centric measures in Social Network Analysis

A small tool for Social Network Analysis, dealing with ego-centric network measures, including Burt's effective size and aggregate constraint and an import code suitable for a large number of adjacency matrices. A free web application is also available on http://www.egonet.associazionerospo.org
