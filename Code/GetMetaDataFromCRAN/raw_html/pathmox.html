pathmox: Pathmox Approach of Segmentation Trees in Partial Least Squares Path Modeling

pathmox, the cousin package of plspm, provides a very interesting solution for handling segmentation variables in PLS Path Modeling: segmentation trees in PLS Path Modeling.
