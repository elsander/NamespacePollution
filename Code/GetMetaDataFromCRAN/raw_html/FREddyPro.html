FREddyPro: Post-Processing EddyPro Full Output File

Despike, ustar filtering, plotting, footprint modelling and general post-processing of a standard EddyPro full output file (LI-COR Inc 2011-2015 <https://www.licor.com/env/products/eddy_covariance/eddypro.html>).
