meta4diag: Meta-Analysis for Diagnostic Test Studies

Bayesian inference analysis for bivariate meta-analysis of diagnostic test studies using integrated nested Laplace approximation with INLA. A purpose built graphic user interface is available. The installation of R package INLA is compulsory for successful usage. The INLA package can be obtained from <http://www.r-inla.org>. We recommend the testing version, which can be downloaded by running: install.packages("INLA", repos="http://www.math.ntnu.no/inla/R/testing").
