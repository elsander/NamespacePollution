autothresholdr: An R Port of the 'ImageJ' Plugin 'Auto Threshold'

Provides the 'ImageJ' 'Auto Threshold' plugin functionality to R users. See <http://imagej.net/Auto_Threshold> and Landini et al. (2017) <doi:10.1111/jmi.12474>.
