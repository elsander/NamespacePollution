twoStageGwasPower: Compute thresholds and power for two-stage gwas

This program computes thresholds and power for a two-stage genome-wide association study. It follows the methods described in Skol AD, Scott, LJ, Abecasis GR, Boehnke M (2006) Nature Genetics doi:10.1038/ng1706 and in the "CaTS" computer program (http://www.sph.umich.edu/csg/abecasis/CaTS/)
