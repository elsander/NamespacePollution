fragilityindex: Fragility Index

Implements and extends the fragility index calculation for dichotomous results as described in Walsh, Srinathan, McAuley, Mrkobrada, Levine, Ribic, Molnar, Dattani, Burke, Guyatt, Thabane, Walter, Pogue, and Devereaux (2014) <doi:10.1016/j.jclinepi.2013.10.019>.
