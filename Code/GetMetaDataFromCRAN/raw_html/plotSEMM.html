plotSEMM: Graphing Nonlinear Relations Among Latent Variables from Structural Equation Mixture Models

Contains a graphical user interface to generate the diagnostic plots proposed by Bauer (2005) and Pek & Chalmers (2015) to investigate nonlinear bivariate relationships in latent regression models using structural equation mixture models (SEMMs).
