GenCAT: Genetic Class Association Testing

Implementation of the genetic class level association testing (GenCAT) method from SNP level association data. Refer to: "Qian J, Nunez S, Reed E, Reilly MP, Foulkes AS (2016) <doi:10.1371/journal.pone.0148218> A Simple Test of Class-Level Genetic Association Can Reveal Novel Cardiometabolic Trait Loci. PLoS ONE 11(2): e0148218".
