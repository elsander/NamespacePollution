gLRTH: Likelihood Ratio Test for Genome-Wide Association under Genetic Heterogeneity

Implements the likelihood ratio test for genome-wide association under genetic heterogeneity as described in Qian and Shao (2013) <doi:10.1111/ahg.12005>.
