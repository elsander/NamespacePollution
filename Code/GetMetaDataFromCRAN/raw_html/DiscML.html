DiscML: DiscML: An R package for estimating evolutionary rates of discrete characters using maximum likelihood

DiscML performs rate estimation using maximum likelihood with the options to correct for unobservable data, to implement a Gamma-distribution for rate variation, and to estimate the prior root probabilities from the empirical data.
