orloca.es: Spanish version of orloca package

Version espanola del paquete orloca que trata el problema de localizacion min-sum, tambien conocido como problema de Fermat–Weber. El problema de localizacion min-sum busca un punto tal que la suma ponderada de las distancias a los puntos de demanda sea minima.
