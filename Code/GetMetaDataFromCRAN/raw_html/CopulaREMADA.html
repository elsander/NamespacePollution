CopulaREMADA: Copula Mixed Effect Models for Bivariate and Trivariate Meta-Analysis of Diagnostic Test Accuracy Studies

It has functions to implement the copula mixed models for bivariate and trivariate meta-analysis of diagnostic test accuracy studies.
