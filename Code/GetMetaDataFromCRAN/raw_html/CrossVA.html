CrossVA: Verbal Autopsy Data Transform for Use with Various Coding Algorithms

Enables transformation of Verbal Autopsy data collected with the WHO 2016 questionnaire for automated coding of Cause of Death using different computer algorithms. Currently supports user-supplied mappings, and provides unvalidated mapping definitions to transform to InterVA4, Tariff 2, and InSilicoVA.
