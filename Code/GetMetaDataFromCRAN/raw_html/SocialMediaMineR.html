SocialMediaMineR: A Social Media Search and Analytic Tool

Social media search and analytic tool that takes one or multiple URL(s) and returns the information about the popularity and reach of the URL(s) on social media. The function get_socialmedia() retrieves the number of shares, likes, pins, and hits on Facebook (<http://www.facebook.com>), Pinterest (<http://www.pinterest.com>), StumbleUpon (<http://www.stumbleupon.com>), LinkedIn (<http://www.linkedin.com>), and Reddit (<http://www.reddit.com>). The package also includes dedicated functions for each social network platform and a function to resolve shortened URLs.
